## Critical Gaps (Match Rate < 70%)

### 1. A-05: Accordion Environment Monitoring (10%)
- **누락**: Sensors.vue의 모바일 per-metric 아코디언 UI 미구현
- **현황**: 그룹 레벨 접기/펼치기만 존재, 개별 센서(온도/습도/CO2 등) 아코디언 없음
- **필요 작업**: Sensors.vue 모바일 뷰에 metric-accordion 컴포넌트 추가

### 2. B-01: Dashboard Widget Customization (40%)
- **누락**: useDashboardLayout.ts 컴포저블은 생성되었으나 Dashboard.vue에 연동 안 됨
- **현황**: 편집 모드 UI, 위젯 표시/숨김, 순서 변경 모두 데드코드 상태
- **필요 작업**: Dashboard.vue에 composable import 및 편집 모드 UI 통합

### 3. A-06: Sidebar Bottom Widget (50%)
- **누락**: 사이드바 위젯 영역에 NotificationCenter만 배치, 장비/알림/룰 수 표시 없음
- **필요 작업**: 온라인 장비 수, 알림 수, 활성 룰 수 카운터 추가

### 4. A-03: Device Card Design Unification (60%)
- **누락**: CSS 변수는 정의되었으나 실제 카드에 data-type 속성/컬러 보더 미적용
- **필요 작업**: Devices.vue 카드에 data-type 속성 + border-top 3px 적용
